# cleaning-origin
## 실행 전 해야할 일
1. npm install로 의존성 패키지 설치
1. npx sequelize db:create로 db 생성
1. .env.example를 복사하여 .env 파일 생성 후, 안에 필요한 변수 설정.(주로 패스워드 설정 해주면 됨.)
